 
# Modul Belajar Python

Ini adalah Modul belajar Python berbasis teks, disini kalian akan belajar mengenai Python dasar
dan diusahakan kalian rajin membaca karena modul ini berbasis teks penjelasan dan beberapa cukup panjang namun tetap mudah dipahami.

## Alur Belajar
Ikuti urutan materi berikut untuk pengalaman belajar yang optimal:
1. **[Dasar-Dasar](./materi/01-dasar)**: Print > Variabel > Tipe Data(dasar terlebih dahulu), Input-output,
dan Operator(aritmetika > incrementDecrement > perbandingan > logika).
2. **[Kontrol Alur](./materi/02-alur)**: Kondisi > looping(whileLoop > forLoop > kontrolLoop)
3. **[Fungsi & Modul](./materi/03-fungsi)**: Fungsi > modul
4. **[Struktur Data](./materi/04-struktur-data)**: Tipe data Struktur.
5. **[Error Handling](./proyek)**: errorHandling
6. **[Proyek Akhir](./proyek)**: Buatlah program kalkulator sederhana
